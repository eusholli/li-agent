# 01-coding.md (Core Coding Standards)

- Adhere to PEP 8 style guide for all Python code.
- Use type hints for function arguments and return values.
- Avoid global variables where possible; prefer passing data as arguments.
- Write clear and descriptive variable and function names.
- Implement error handling using `try-except` blocks for potential issues.
- Ensure all functions are testable and follow single responsibility principle.
